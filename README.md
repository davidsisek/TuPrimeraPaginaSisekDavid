
La App Baquedano describe un negocio situado en la ciudad de Pergamino, Provincia de Buenos Aires
Este negocio se dedica a satisfacer las necesidades de 3 importanes rubros:
    Confeccion de Toldos
    Venta de Ropa de Trabajo
    Venta de Articulos de Camping

Para este desarollo se buscó proporcionar un sistema que le permita:
    Registrar presupuestos de Toldos
    Stock de Articulos de Camping

La tabla de toldos se divide en:
    Cliente
    Tipo de Toldo
    Precio

La tabla de Artículos de Camping se divide en:
    Nombre del Articulo 
    Precio

PRUEBA DE FUNCIONAMIENTO
1. Correr el comando python manage.py runserver
2. Una vez abierto el link que se proporciona se mostrara la pagina diseñada
3. Alli deberan registrarse con un usuario
4. Luego loguearse y poder acceder al home
5. Las pestañas de Toldos y Camping sirven para controlar el stock de los articulos,
se pueden agregar nuevos, editar y borrar los mismos.
6. Tambien se puede acceder al perfil y editarlo y tambiena agregar una imagen para agregar al avatar
7. Se puede acceder al perfil de admin para visualizar los modelos, articulos y usuarios
